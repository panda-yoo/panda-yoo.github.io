# Quantum Kernel SVM

**Repository:** [Quantum Kernel SVM Project](https://github.com/panda-yoo/Quantum-kernel-SVM-Project)

A quantum machine learning project investigating **quantum kernel-based Support Vector Machines (QSVMs)** by comparing multiple quantum feature-map circuits across synthetic and real-world datasets.

Implemented using **Qiskit** and **scikit-learn**, this project studies how quantum feature map design and the encoding parameter **α** affect classification accuracy, decision boundaries, and kernel performance.

Developed as part of **PH874 – Basics of Quantum Computing**.

---

## Project Motivation

Kernel methods classify data by implicitly mapping it into higher-dimensional feature spaces. Quantum computing offers a natural framework for such embeddings using Hilbert spaces generated by quantum circuits.

This project explores whether parameterized quantum feature maps can improve classification performance over classical approaches.

---

## Project Objectives

- Compare multiple quantum feature-map circuits
- Study how encoding parameter α affects performance
- Evaluate QSVMs on synthetic and real datasets
- Compare quantum kernels against classical SVM kernels
- Visualize decision boundaries

## 1. QMLproject.ipynb

Focused study comparing quantum and classical kernels on the Blood Transfusion dataset.

### Workflow

- Load dataset
- Select weakly correlated features:
  - Recency (months)
  - Time (months)
- Normalize features to `[0, π]`
- Build custom quantum feature maps
- Generate kernel matrices
- Train `SVM(kernel="precomputed")`
- Compare against classical:
  - RBF kernel
  - Linear kernel
  - Polynomial kernel

### Outputs

- Quantum circuit visualizations
- Quantum decision boundaries
- Classical decision boundaries
